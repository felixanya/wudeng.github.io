付费

  • ARPU
    • Average Revenue Per User
    • 人均充值，有的是付费用户，有的是所有用户
    • 大多数网游公司趋向于采用：月收入 / 月活跃付费用户数，所以更标准的叫法是ARPPU
  • ARPPU
    • Average Revenue per Paying User
    • 每付费用户的平均收益
    • 总收入 / APA
  • APA
    • Active Paid Account
    • 活跃付费账户
  • PUR
    • Pay User Rate
    • 付费比
  • LTV
    • Life Time Value
    • 生命周期总价值
  • ROI
    • Return On Investment
    • LTV / CPI * 100%

成本CP*

  • CPC
    • (每点击成本)
  • CPI
    • (每安装成本)
  • CPA
    • (每激活成本)，
    • LTV > CP*时，渠道投放才有价值

在线玩家数量

  • PCU
    • Peak Concurrent Users
    • 峰值在线人数
  • ACU
    • 平均在线人数
    • Average Concurrent Users

活跃玩家数量

  • DAU
    • 日活跃
    • Daily Activy Users
  • MAU
    • 月活跃
    • Month Activy Users
  • DNU
    • Daily New User
