## sudoers
对应的文件为`/etc/visudoers`，使用visudo进行修改。
`%group`

```
%admin  ALL=(ALL) ALL
```
修改为：
```
%admin  ALL=(ALL) NOPASSWD:ALL
```

查看当前系统语言环境：

locale

```
LANG="en_US.UTF-8"
LANGUAGE="en_US:en"
```

vim ~/.bashrc
```
export LC_TYPE=en_US.UTF-8
export LC_ALL=en_US.UTF-8
```
